# Convolve

A calculator to perform convolution operation on matrices.

# Demo

[https://convolver-three.vercel.app/](https://convolver-three.vercel.app/)

# How to use

- Install dependencies

  ```sh
  yarn install
  ```

- Run in dev mode

  ```sh
  yarn dev
  ```

- Build the app

  ```sh
  yarn build
  ```
